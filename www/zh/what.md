# ByteMind 是什么？

ByteMind 是一个用 Go 实现的**终端优先 AI 编程助手**。

它让你在代码仓库中直接与大语言模型协作 — 读取文件、进行修改、执行搜索和运行命令 — 全程不离开终端。

## 它解决的问题

当下大多数 AI 编程工具依赖浏览器、云端 IDE 或专有编辑器插件。ByteMind 采用不同思路：

- **留在终端。** 无需在编辑器和浏览器之间来回切换。
- **代码保持本地。** ByteMind 直接操作你的本地文件系统，除你配置的 API 调用外，代码不会上传至第三方服务。
- **完成完整任务。** 不只是代码建议 — ByteMind 能在多步循环中读取、修改、搜索并验证变更，直到任务完成。

## 工作原理

在你的代码仓库中启动一个聊天会话：

```bash
bytemind chat
```

ByteMind 从工作区收集上下文，发送给你配置的 LLM，并进入工具调用循环：

1. 模型读取文件、执行搜索、提出变更建议。
2. 修改文件系统或执行 Shell 命令的操作需要你的明确批准。
3. 循环持续到任务完成或达到迭代预算上限。
4. 所有内容保存至可随时恢复的持久化会话。

## 设计原则

| 原则 | 含义 |
|------|------|
| **本地优先** | 在你的机器上运行，无强制云端依赖 |
| **终端优先** | 原生 CLI 与 TUI，无需 GUI |
| **Provider 无关** | 兼容 OpenAI 接口协议与 Anthropic |
| **用户掌控** | 审批机制与迭代预算确保你始终在掌控之中 |
| **可恢复** | 所有会话持久化保存，随时可恢复 |
